# Ender IO Grinding Ball Additions for ATM10

This mod adds new grinding ball tiers — Allthemodium, Vibranium, Unobtanium, and Infinite — to Ender IO, specifically designed for the All the Mods 10 (ATM10) modpack. These grinding balls enhance machine performance by significantly boosting speed, efficiency, and durability, allowing for faster processing times and more efficient workflows.

## Features

### Allthemodium Grinding Ball:

- Increases machine speed and efficiency.
- Perfect for early-tier upgrades in ATM10.
- Standard durability for reliable performance.

### Vibranium Grinding Ball:

- Further enhances machine speed and capacity.
- Ideal for mid-game setups requiring higher efficiency.
- Improved durability over Allthemodium.

### Unobtanium Grinding Ball:

- Maximum performance boost with superior processing speed and capacity.
- Designed for late-game automation and large-scale processing.
- Exceptional durability for extended use.

### Infinite Grinding Ball:

- Ultimate performance upgrade with infinite durability.
- Maximum speed, efficiency, and capacity multipliers.
- The pinnacle of grinding ball technology.

## Compatibility

- Fully integrated into Ender IO's grinding system.
- Balances seamlessly with ATM10 progression, leveraging custom alloys and resources.
- Works seamlessly with other Ender IO addons and mods.

## Dependencies

This mod requires the following to function:

- **Ender IO**: Provides the base framework for machines, grinding balls, and the core grinding system.
- **Ender IO Base**: Required for grinding ball API and machine integration.
- **Allthemodium**: Supplies the resources necessary to craft the new grinding ball tiers, including Allthemodium, Vibranium, and Unobtanium.

## Installation

- Download and install the ATM10 modpack.
- Add this mod to your mod folder.
- Launch the game and enjoy enhanced machine performance!

## Configuration

All grinding ball properties are fully configurable:

- **Speed**: Adjust processing speed multipliers
- **Efficiency**: Modify energy efficiency per tier
- **Unbreaking**: Control durability decay rates
- **Durability**: Set maximum uses per grinding ball

Edit the config file in `config/grindingballadditions-common.toml` to customize values to your preference.
